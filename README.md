## 💫 About Me

Saya adalah mahasiswa **Informatika di Universitas Papua (UNIPA)** yang sangat antusias dengan dunia *software engineering*, terutama di bidang pengembangan web. Saya tidak hanya belajar teori, tapi juga aktif mengimplementasikannya ke dalam proyek-proyek yang bermanfaat.

- 🔭 Saat ini sedang berfokus mendalami ekosistem **Full-Stack Development (MERN, FERN, Go, Next.js)**.
- 🚀 Sedang dan telah berkontribusi pada beberapa proyek seperti **Sistem E-Voting HIMAKOM UNIPA**, **Sistem Manajemen Website KSK**, hingga proyek **Kooperasi Gereja**.
- 🎨 Selain *coding*, saya juga memiliki ketertarikan kuat dalam **UI/UX Design** dan **Network Administration**.
- 👯 Selalu terbuka untuk berkolaborasi dalam proyek *open-source* atau pembuatan website inovatif.
- ⚡ **Fun fact:** Selain memecahkan *bug*, saya menikmati waktu dengan bermain game (Valorant/Project Zomboid/Steam/PUBG Mobile) dan fotografi/desain!
